# Charlie: Embodied AI Humanoid Robot for Interactive Education

Charlie is an AI-powered social humanoid robot developed for interactive education. Built upon the open-source Poppy humanoid platform, Charlie integrates conversational AI, expressive facial animation, autonomous body gestures, and real-time question answering to create engaging classroom experiences.

---

## Features

- AI-powered educational assistant
- Interactive lesson delivery
- Real-time question answering
- Animated digital face
- Autonomous arm and head gestures
- Voice interaction
- Raspberry Pi based controller
- Dynamixel servo motor control

---

## Hardware

- Raspberry Pi 4B
- Dynamixel MX-28AT Servo Motors
- Dynamixel AX-12A Servo Motors
- Robotis U2D2 Interface
- USB Microphone
- Bluetooth Speaker
- LCD Face Display
- USB Camera

---

## AI Stack

- GPT Realtime API
- GPT-4o Mini Transcribe
- GPT-4o Mini TTS
- Python
- WebSockets
- Dynamixel SDK
